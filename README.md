# Hugo theme Mage

A Hugo theme features soft color scheme, card-based layouts and a flexible switching dark/light theme.

**Be aware:** This project is still under development, changes may occur at anytime.

## Usage

### use git clone

Use `git clone` to clone this repository to your local project, you can customize this theme if you want.

### use as a git submodule

**NOTICE:** There is no way you can modify the theme and push the changes in this way.

To use this theme as a git submodule,

First, inside your project directory, run:
```shell
git submodule add https://github.com/mengshitia/hugo-theme-mage.git themes/hugo-theme-mage
```

then adding this line in your site configuration:
```toml
theme = 'hugo-theme-mage'
```

All done.

See also: [How to use a git submodule](https://git-scm.com/book/en/v2/Git-Tools-Submodules).
